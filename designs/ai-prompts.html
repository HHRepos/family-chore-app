Living Garden · AI image prompts · v1.0

Prompts you can copy.

Every visual asset for the Living Garden Phase 2 — trees, flowers, skies, characters, wildlings, scenes — has a tuned AI image prompt below. Tap Copy prompt, paste into Midjourney / DALL-E / Krea / Ideogram / Stable Diffusion, save the result with the suggested filename, drop into the folder. Each prompt is locked to the watercolour-Ghibli aesthetic from the designer brief so the output stays consistent across ~30 generations.

Workflow: Generate the foundation tree (`tree-day7.png`) first. Confirm the style feels right. Then go through the rest in any order — the prompts share enough style language that the output will be coherent. Re-roll any image until it matches the foundation's vibe.

Style consistency rules

Every prompt above ends with the same negative-prompt list. Repeat these in each generation for visual consistency. If your AI tool has a "negative prompt" field, paste this in:

universal-negatives paste in every "negative prompt" field

And a quick recap of the visual rules from the designer brief:

RuleWhy
Soft edges, no hard outlines Watercolour, not vector. Implied stroke through colour transitions.
Pastel base, warm amber accents Sky 30-40% sat, accents 70-80% used sparingly.
Dappled warm light from above Always feels like sunlight through canopy.
Magic in the margins Fireflies, dust motes, leaves drift past — never centre stage.
Painterly, not vector-perfect Slight texture, hand-finished feel.
One typeface, two weights Doesn't apply to images, but applies if any UI text gets generated. Don't.

What we're NOT making

  • Not gamified — no XP bars, no levels visible, no badge collections in the art
  • Not cartoonish — characters have personality but they're painted, not bouncy
  • Not minimal/Scandinavian — soft and rich, not flat and geometric
  • Not retro / pixel-art / 3D / vector / commercial-clipart

Once you have all the files

  1. Save each generated image with the filename shown in the prompt card.
  2. Drop into family-chore-app/garden-art/ in the matching subfolder.
  3. Commit them: git add garden-art/ && git commit -m "Add Phase-2 illustrator pack"
  4. Tell me when they're in. I'll wire them into the SwiftUI views and ship Build 17 with real visuals.
